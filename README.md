## Installation

### Homebrew

```sh
$ brew install koki-develop/tap/todoist-cli
```

### go install

```sh
$ go install github.com/koki-develop/todoist-cli@latest
```

### Release

Download the binary from the [releases page](https://github.com/koki-develop/todoist-cli/releases/latest).
